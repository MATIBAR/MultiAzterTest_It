Kasu hiperbolikoan, Cauchy- ren datuak emateko erabili dugun lerroa inon karakteristiken tangentea ez bada, soluzio lokala existituko da. Baina Cauchy lerroa eta karakteristika bat puntu batean tangenteak izango balira, arazoak sortuko lirateke. 4.- Adibideak Uhin ekuazioa c 2u xx − utt = 0 , A = c 2 ; B = 0 ; C = −1 ; ∆ = B 2 − AC = c2 > 0 ; karakteristiken ekuazioa : c 2 dt ds 2 − dx ds 2 = 0 ; karakteristikak : x − ct = ξ , x + ct = η . ekuazioa, aldagai berrietan : uξη = 0 , soluzio orokorra : u(x, t) = f+(x − ct) + f−(x + ct) . x 2u xx − utt = 0 , A = x 2 ; B = 0 ; C = −1 ; ∆ = B 2 − AC = x2 > 0 (x = 0 lerroan izan ezik) ; karakteristiken ekuazioa : x 2 dt ds 2 − dx ds 2 = 0 ; karakteristikak : ln x − t = ξ , ln x + t = η . ekuazioa, aldagai berrietan : uξη = 1 4 (uξ + uη) . 4  5.- Ekuazio hiperbolikoak: karakteristikak Informazioa edo singularitateak karakteristiketan zehar abiatzen dira. Karakteristikak ondoko sistema endakatua izateko baldintzetik datozkigu: ux + uy = 0 , du = uxdx + uydy = 0 . Kasu honetan: 1 1 dx dy = dy − dx = 0 . Ondorioz  (kasu honetan), du = 0 karakteristiken gainean, eta u ez da aldatzen karakteristiketan. Karakter- istiken ekuazioa: ξ = x − t. Soluzioa: u(x, t) = f (x − t). Orokorrean: Ekuazioa : A(x, y, u)ux + B(x, y, u)uy = C(x, y, u) ; aldaketa : uxdx + uydy = du ; endakapen baldintza : A B dx dy = Ady − Bdx = 0 ; ondoriokoa : dx A = dy B = du C . Adibidea: Ekuazioa : ux + uy = 1 ; aldaketa : uxdx + uydy = du ; endakapen baldintza : 1 1 dx dy = dy − dx = 0 ; ondoriokoa : dx = dy = du , soluzioa : Φ(x − y, u − x) = 0 ⇔ u(x, y) = x + f (x − y) . Problema: Ekuazioa : yux − xuy = 0 ; baldintza : soluzioa x = 0 , u = y 2 lerrotik pasatzen da ; aldaketa : uxdx + uydy = du ; endakapen baldintza : y −x dx dy = ydy + xdx = 0 ; ondoriokoa : dx y = dy x = du 0 , ek. soluzioa : Φ(x 2 + y2, u) = 0 ⇔ u(x, y) = f(x2 + y2) ; problemaren soluzioa : u(x, y) = x 2 + y2 . 6  7.- Eremu bektorialak F(x, y, z) eremu bektorialaren eremu lerroak edo lerro bektorialak:  puntu bakoitzean paraleloak. F = Aˆi + Bˆj + Cˆj bektore eremuaren eremu lerroak Azx + Bzy = C ekuazioaren lerro karakteristikoak dira. Gainazal bektoriala: lerro bektorialek osatutako gainazala. Deﬁnizio orokorra:  normala eta F orto- gonalak dira. Beraz, ek. diferentzialaren soluzioa. Adibidez: xzx − yzy = z , eta soluzioaren gainazalean y = 1, z = 3x lerroa dago. Karakteristikak: dx x = − dy y = dz z . Soluzioak: xy = c1, zy = c2. Soluzio orokorra: Φ(xy, yz) = 0 ⇔ z(x, y) = φ(xy)/ y. Dagokiona: y = 1 z = 3x ⇒ x = s y = 1 z = 3s → xy = c1 → c1 = s zy = c2 → c2 = 3s → c2 = 3c1 → yz = 3xy → z(x, y) = 3x .
