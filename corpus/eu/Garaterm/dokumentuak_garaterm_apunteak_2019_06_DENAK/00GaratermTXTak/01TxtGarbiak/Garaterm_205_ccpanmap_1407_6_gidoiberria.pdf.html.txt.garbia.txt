   Topologia Orokorra 1. Espazio Topologikoak 1 Espazio Topologikoak Aurreko kurtsoan ikusi duzuenez, espazio metrikoen arteko jarraitasunari bu- ruz hitz egin ahal izateko, nahikoa da bi espazioen irekiak zein diren jakitea. Gogoratu: Bi espazio metrikoen arteko aplikazio bat jarraikia da baldin eta soilik baldin bigarren espazioaren edozein irekiren aurreirudia lehenengo espazioan irekia bada. Hori dela eta, jarraitasunari buruz hitz egiteko, zera pentsa dezakegu:  hasierako nozioa ez duela zertan distantzia izan behar, baizik eta irekien familia. Horrexegatik espazio metriko baten irekien propietateak gogoratuko ditugu: - Multzo hutsa eta totala irekiak dira. - Irekien bildura (edozein!) irekia da. - Irekien ebakidura ﬁnitua irekia da. Espazio metrikoetan lehenengo biak betetzen direla egiaztatzea berehalakoa da. Hirugarrena frogatzeko bakarrik erabili behar da zentro bereko bola irekien eba- kidura ﬁnitua puntu berean zentratutako beste bola ireki bat dela. Orain demagun aurreko hiru propietateak betetzen dituen azpimultzoen fa- milia bat daukagula (familia honek orain ez du zertan espazio metriko baten irekien familia izan behar). Edozein kasutan, oraindik ere deﬁnitu ahal izango ditugu aurreko kurtsoan ikusitako nozio gehienak. Eta horixe egingo dugu. Irekien familiak jarraitasuna zehazten dutenez eta ondorioz itxura ere bai, irekiek topologia (= itxuraren zientzia) bat osatzen duela esaten da. Guretzat topologia irakasgaiaren izena (Topologia Orokorra) izateaz gain, kurtsoan zehar erabiliko dugun oinarrizko nozioa. Beraz lehenengo gai honetan kurtsoaren oinarrizkoa nozioak aurkeztuko di- tugu: azpimultzoak irekiak izango dira eta azpimultzoen familia topologia. Ho- rrez gain beste nozio batzuk ere aztertuko ditugu: azpimultzo irekiak, ireki- oinarriak, azpionarriak eta Hausdorﬀ-en espazioak. 1.1 Deﬁnizioa eta zenbait adibide Deﬁnizioa 1.1. Izan bedi X multzoa eta τ ⊆ P(X). τ X-ren gainean deﬁ- nitutako topologia dela esango dugu baldin eta ondoko hiru axiomak betetzen baditu: (τ1) ∅, X ∈ τ . (τ2) τ -ren elementuren edozein familia ﬁnituren ebakidura τ -n dago. (τ3) τ -ren elementuren edozein familiaren bildura τ -n dago. τ X-ren gainean deﬁnitutako topologia emanda, (X, τ ) bikoteari espazio topologikoa deritzo. Topologiaren elementuak azpimultzo irekiak izango dira. Javier Guti´ errez 1 2009-10 ikasturtea  Topologia Orokorra 1. Espazio Topologikoak Adibideak 1.2. (i) (R n, du) espazio euklidearrean azpimultzo irekien familia kontsideratzen badugu R n-ren gainean deﬁnitutako topologia bat izango dugu. Topologia honi ohiko topologia deritzo eta (R n, τu) idatziko dugu. (ii) Izan bedi X multzoa. τDisk = P(X) topologia diskretua da. (iii) Izan bedi X multzoa. τIndisk = {∅, X} topologia indiskretua da. (iv) Izan bedi X = {a, b}. τSier = ∅, {a}, X Sierpinski-ren topologia da. (v) Izan bedi X multzo inﬁnitua. τKof = {A ⊆ X : X − A ﬁnitua} ∪ {∅} topologiari topologia koﬁnitua deritzo. Ariketa 1.i. τ = {A ⊆ X : A ﬁnitua } ∪ {X} topologia al da? Ariketa 1.ii. Zein da τKof X ﬁnitua denean? (vi) Izan bedi X multzo ez-kontagarria. τKok = {A ⊆ X : X − A kontagarria} ∪ {∅} topologiari topologia kokontagarria deritzo. Ariketa 1.iii. Zein da τKok topologia X kontagarria denean? (vii) Izan bitez X multzoa eta A ⊆ X. τA = {B ⊆ X : A ⊆ B} ∪ {∅} topologiari topologia A-sartuta deritzo. Ariketa 1.iv. Zein da τA A = ∅ eta A = X denean? (viii) Izan bitez X multzoa eta A ⊆ X. τ A = {B ⊆ X : A ∩ B = ∅} ∪ {X} topologiari topologia A-baztertuta deritzo. Ariketa 1.v. Zein da τ A topologia A = ∅ eta A = X denean? (ix) τKol = {∅, R} ∪ (a, +∞) : a ∈ R topologiari Kolmogorov-en topologia deritzo. (R, τKol) Kolmogorov-en zuzena izango da. (x) τScat = {U ⊆ R : U = A ∪ B, A ∈ τu eta B ⊆ I} topologiari Scattered topologia deritzo. (R, τScat) Scattered zuzena izango da. Deﬁnizioa 1.3. Izan bitez τ1 eta τ2 X-ren gainean deﬁnitutako bi topologia. τ1 τ2 baino ﬁnagoa dela esango dugu τ2 ⊆ τ1 bada. τ1 eta τ2 konparagarriak direla esango dugu baldin eta τ1 τ2 edo τ2 τ1 baino ﬁnagoa bada. Ariketa 1.vi. τKol, τScat eta τu aurreko adibideetan deﬁnitutako R-ren gaineko topologiak izanik. Zein da zein baino ﬁnagoa? Ariketa 1.vii. Izan bitez τ1 eta τ2 X-ren gainean deﬁnitutako bi topologia. Izan bedi τ = τ1 ∩ τ2 = {U ⊆ X : U ∈ τ1 eta U ∈ τ2}. Frogatu τ topologia dela eta τ1 eta τ2 topologiak τ baino ﬁnagoak direla. Izan bedi τ ∗ X-ren gainean deﬁnitutako beste topologia bat non τ1 eta τ2 to- pologiak τ ∗ baino ﬁnagoak diren. Frogatu τ ere τ ∗ baino ﬁnagoa dela. (X, τ ) espazio topologikoaren itxien familia normalean C idatxiko dugu, hau da C = {F ⊆ X : X − F ∈ τ }. Oharra 1.5. Oso errore tipikoa da C = P(X) − τ betetzen dela pentsatzea, hau da, azpimultzo bat itxia del baldin eta irekia ez bada. Kontuz! Ariketa 1.viii. Aurkitu Scattered topologian ((x) adibidea) azpimultzo bat irekia eta itxia batera dena (∅ eta R ez dena). Teorema 1.6. (X, τ ) espazio topologikoan azpimultzo itxiek ondoko propieta- teak betetzen dituzte: (C1) ∅ eta X itxiak dira. (C2) Azpimultzo itxien ebakidura azpimultzo itxia da. (C3) Azpimultzo itxien bildura ﬁnitua azpimultzo itxia da. Izan ere, hiru propietate hauek betetzen dituen familia bat emanda, C, orduan τ = {X − F : F ∈ C} topologia izango da. Ariketa 1.ix. Izan bedi X multzoa. Frogatu C = {F ⊆ X : F ﬁnitua} ∪ {X} familiak (C1), (C2) eta (C3) propietateak betetzen dituela. Zein da kasu honetan τ = {X − F : F ∈ C} topologia? Ariketa 1.x. Izan bedi C = {F ⊆ R : F = G ∩ H, G ∈ Cu eta Q ⊆ H}. Frogatu (C1), (C2) eta (C3) propietateak betetzen dituela. Izan bedi τ = {X − F : F ∈ C} topologia. Aztertu ondoko azpimultzoen artean zein diren irekiak edota itxiak: A1 = Q, A2 = I,  , τu konexua denez aurrekoa ezinezkoa da eta beraz absurdu batera heltzen gara.
