SEINALIZAZIO-SISTEMA BIOLOGIKOAK (Bioeraentzaileka)
Animalien funtzionamendu integratua nerbio-sistemaren eta sistema endokrinoaren
kontrol pean dago

-Nerbio-sistema-------Neurotransmisoreak
-Sistema endokrinoa---Hormonak

Zelulen arteko seinalizazio barianteak
• Autokrino: zelula berera
• Parakrino: hurbil dauden zelulen arteko difusioa gune interstizialetik
• Neuronal: sinapsis neuronen artean
• Neuroendokrino: neuronatik zirkulazio sistemara
• Endokrino: Guruin zelulatik odol bitartez itu zelulara

    * Sistema endokrinoak honako prozesu biologiko kontrolatzen ditu:
          o Kolore-aldaketak
          o Ugalketa: sexuaren finkapena, gonadaren garapena,…
          o Hazkuntza
          o Homeostasia: metabolismoa, ormoeraenketa,…
          o Metamorfosia eta muda
          o …

HORMONAK
    * Organo edo ehun batek ekoiztutako substantziak, organismoaren beste
      eskualde batean jarduteko barne-medioan garraiatu egiten direnak
      (Jatorria: neurona neurojariatzaileen produktuak)
    * Osaketa kimikoa:
          o Peptidoak: animoazidoak edo péptidos txikiak
          o Lipidoak: esteroideoak batik bat
    * Hormona itu-zelula ezagutu behar du:
          o PEPTIDIKOAK-MINTZ-HARTZAILEAK
          o ESTEROIDEOAK-HARTZAILE ZITOSOLIKOAK
    * Guruin endokrinoetan edo zelula jariatzaile isolatuetan produzitzen dira:
          o GURUIN SOKAKARAK
                # GURUIN FOLIKULARRAK
                # NEURONA NEUROJARIATZAILEAK
          o ZELULA JARIATZAILEAK:
                # JARIAKIN PROTEIKOZKOAK
                # JARIAKIN ESTEROIDEOZKOAK

    * KRUSTAZEOAK
          o X-organoa (neurojariatzailea, MIH sortu, jariatu)
          o Barrunbe guruina (Neurohemalea, MIH askatu, hemolinfara)

INTSEKTUEN SISTEMA ENDOKRINOA
    * Hormona diuretikoa
          o Gernuaren ekoizpena kitzikatu
          o Zerebroko zelula neurojariatzaileetan sintetizatu
          o Axonen bidez corpora cardiaca-ra garraiatu
          o Sistema zirkulatoriora askatu (organo nurohemala)
    * Hormona adipozinetiko
          o Gantz-gorputzeko erreserba lipidikoak mobilizatu
          o Corpora cadiaca-ko zelula neurojariatzaileak ekoiztu
          o Sistema zirkulatoriara askatu (organo neurohemala)
    * Hormona protorazikoa (PTTH)
          o Mudaren prozesua eraendu
          o Zerebroko zelula neurojariatzeetan sintentizatu
          o Axonen bidez corpora cardiaca-ra garraiatu
          o Ekdisonaren sisntesia kitzikatu guruin protorazikoan
    * Gazte-hormona (JH)
          o Hazkuntza eta metamorfosia prozesuak eraendu
          o Corpora allata-ko zelula neurohariatzeetan sintetizatu
          o Sistema zirkulatoriara askatu (organo neurohemala)
