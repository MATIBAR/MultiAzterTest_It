1. ZENBAKI KONPLEXUAK
Gai honetan zenbaki konplexuen multzoaren oinarrizko propietateak aztertzen dira.

Zenbaki konplexuen multzoa.
Zenbaki errealen bikote ordenatuak dira zenbaki konplexuak. Beraz, mul-
tzo modura ikusita R2-ren berbera dugu.
Notazioa eta terminoak.
• Zenbaki konplexuen multzoa C da.
• (x, y) bikotea x+yi idazten da. Idazkera honi binomio erakoa deritzo.
• x + 0i zenbaki konplexua x zenbaki errealarekin identiﬁkatzen da;
horrela, R ⊂ C dugu.
• z = x + yi bada, x da z-ren parte erreala ( z) eta y, z-ren parte
irudikaria ( z).
• Parte erreal nulua duten zenbakiei irudikari hutsak deritze.
Eragiketak.
• Batuketa: (x1 + y1i) + (x2 + y2i) = (x1 + x2) + (y1 + y2)i.
• Biderketa: (x1 + y1i) · (x2 + y2i) = (x1x2 − y1y2) + (x1y2 + x2y1)i.
Eragiketa bi hauekin C-ren egitura algebraikoa gorputz trukakorra da.
Gogoratu (edo aurkitu) gorputz trukakorra izateko behar diren propietateak eta
egiaztatu betetzen direla.
Plano konplexua.
C eta R2 identiﬁkatzen direnez, zenbaki konplexuak planoko puntuekin
identiﬁkatzen dira.
Modulua eta argumentua. Forma polarra.
z = x+yi zenbaki konplexuari dagokion planoko (x, y) puntuaren koorde-

dira. Batzuetan komeni da argumentu guztietatik bat aukeratzea, (−π, π]
tartean dagoen bakarra; argumentu nagusia deritzo.

2. FUNTZIO ANALITIKOAK
Gai honetan aldagai konplexuko funtzio baten deribatua deﬁnituko dugu. Aurre-

badira ere, deribazioak propietate bereziak eskatzen dizkio funtzioari.
Aldagai konplexuko funtzio konplexuak.
D ⊂ C bada eta f : D ⊂ C → C funtzioa badugu, f aldagai konplexuko
funtzio konplexua da.
Funtzio izateko, z ∈ D bakoitzerako f (z) bakarra da. Geroago interesgarria
izango da kasu batzuetan f (z)-

horretan funtzio multiformea deitzen da.
Bektore-funtzio batekiko identiﬁkazioa. f (x+iy) = u(x, y)+iv(x, y) bada,
u eta v errealak izanik, f eta (x, y) ∈ D → (u(x, y), v(x, y)) ∈ R2 funtzioa
identiﬁka daitezke.
Distantzia eta topologia plano konplexuan.
Planoko distantzia euklidearra hartzen da C-rako: d(z1, z2) = |z1 − z2|.
Horrekin C espazio metriko bihurtzen dugu. Espazio metriko modura C eta
R2 identiﬁkatuta daude.
Kontzeptu topologikoak. Espazio metrikoetako zenbait kontzeptu gogoratu
behar dira: distantzia, ingurune, barruko, kanpoko eta mugako puntuak; multzo
irekia, itxia; metatze-puntuak; multzo trinkoa; multzo konexuak eta arkuz kone-
xuak.
Emaitza pare bat: (i) C-ko multzo bat trinkoa da baldin eta soilik baldin itxia

Gai honetan kurben gaineko integralak deﬁnituko ditugu funtzio konplexu jarrai-

tza nagusia Cauchyren teorema integrala da:
 kurba sinple itxi baten gainean inte-

formula integralaren bidez.
Ondorio bat izango da funtzio analitikoek behin eta berriro deriba daitezkeela eta
deribatuetarako ere formula integralak daude. Beste ondorio batzuk ere baditu
formula integralak.
Kurbak.
• Kurba parametrizatua: γ : [a, b] → C aplikazio jarraitua. Baldin

• γ(a) hasierako puntua da eta γ(b) amaierako puntua.
