MOTIBAZIOAREN TEORIAK:

Kirol eta jarduera fisikoaren inguruan gehien erabiltzen diren ikuspuntu

teorikoak:

Lorpenaren teoria: bakoitzaren izaerak eta egoerak izango du eragina

∙ Kirolak -> lorpenaren izaera. Kirolariak euren burua gainditzen saiatzen dira

∙ Aldagai pertsonal eta egoerazkoen menpe

∙ Gizakiok kontrajarritako bi indar ditugu:

- Garaipena lortzeko motibazioa

- Porrota ekiditeko motibazioa

∙ Bietatik garrantzitsuenak markatuko du pertsonaren lorpen motibazioa

∙ Goi mailako kirolariak: harroagoak => arrisku egoerak aukeratu eta erronkak

bilatzen dituzte (garaipena lortzeko motibazioa)

∙ Behe mailetako kirolariak: porrotarekiko beldurrean oinarritu => arrisku egoerak

ekidin (porrota ekiditeko motibazioa)


Ebaluazio kognitiboaren teoria (barne eta kanpo motibazioaren teoria):

∙ Pertsonek egiten dituzten jarduerek euren barne eta kanpo motibazioan eta

motibazio ezan dituzten eraginak azaltzen du

∙ Kirolariak bere jardueran trebea sentitu behar du; horrela ez bada -> barne m.

gutxitu eta desmotibazioa igo

∙ 3 motibazio mota (bakoitzak aukeratzen dituen jardueren arabera):

- Barne motibazioa

- Kanpo motibazioa

- Desmotibazioa

∙ 3 orokortasun maila daude zeinean motibazio motek elkar eragiten duten eta

pertsonak duen motibazio mota zergatik duen azaltzen den:

- Maila orokorra: nortasunaren arabera motibazio ezberdina izateko joera

izango du

- Testuinguruaren maila: pertsona bakoitzak inguru ezberdinak ditu (familia, lagunak …) => bizitzaren arlo ezberdinetan motibazioa ere ezberdina izango

da

- Egoeraren maila: egun ezberdinetan ez dugu motibazio =a gauza =a egiteko

orduan. Egoeraren arabera motibazioa aldatuko da

∙ Orokortasun mailen eta motibazio mailen arabera -> motibazio aldaketak

azaldu => jarduerak motibazioan izango dituen ondorioak aurreikus daiteke

∙ Teoria honekin motibazioaren aldaketak aztertzen dira

Atribuzio kausal eta emozionalen teoria:

∙ Atribuzio kausala: gertatzen zaigunari ematen diogun azalpena -> zergatik egin

dugun gauza bat ongi / gaizki

∙ Egoeraren aurrean kirolariak izango dituen itxaropen, joera eta jarreran izango

du eragina era determinantean -> atribuzio kausalek eragina izango dute

∙ Berez gertatuko da edozein ondorioen aurrean, batez ere eragina negatiboa

denean

∙ Sailkapena -> 3 dimentsioren araberakoa:

- Barnekoa (tuntuna naiz) / kanpokoa (irakasleak mania dit)

- Egonkorra (tuntuna) / ezegonkorra (zorte txarra / ona)

- Kontrolagarria (gehiago / gutxiago ikastea) / ez-kontrolagarria

∙ Atribuzio bakoitza -> emozio batekin erlazionatuta

∙ Atribuzioak desegokiak direnean -> hitzezko pertsuasioarekin aldatu behar dira
