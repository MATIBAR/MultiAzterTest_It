2.2.2. PROBABILITATE-BANAKETA

Demagun X: (Ω, F, P) → (R, B) zorizko aldagaia dela. Orduan:

∀B∈B: X-1(B) = (X∈B) ∈ F

X-ren probabilitate-banaketa ondoko erara definitzen da:

PX : B → [0, 1]

B → PX(B) = P(X∈B) = P(X-1(B))

X-ek B-n balioak hartzeko probabilitatea neurtzen du.


Teorema: Demagun X: (Ω, F, P) → (R, B, PX) zorizko aldagaia dela. Orduan PX probabilitate-banaketa (R, B) espazioan definituriko probabilitate-neurria da.



Oharra: X eta Y zorizko aldagaiak probabilitate-banaketa berdina eduki dezakete, nahiz eta probabilitate-espazio ezberdinetan definiturik egon.



Demagun X:(Ω, F, P)→(R, B, PX) eta Y:(Ω*, F*, P*)→(R, B, PY) probabilitate-espazio ezberdinetan definituriko zorizko aldagaiak direla. Orduan X eta Y berdinki banatuta daudela esango dugu, ondokoa betetzen baldin bada:

∀B∈B: PX(B) = PY(B)




Gehienetan, B boreldar guztiak ez direnez ezagutzen, probabilitate- banaketa ezin da erabili praktikoki. Horrexegaitik probabilitate-banaketa zehazten duen funtzioa definituko dugu, baina boreldar batzuentzako soilik.

Demagun X: (Ω, F, P) → (R, B, PX) zorizko aldagaia dela. Orduan, FX

ondoko funtzioa X-en banaketa-funtzioa deitzen da:

x → FX(x) = PX(-∞, x] = P(X ≤ x)

bere banaketa-funtzioa direla. Orduan, ondoko propietateak betetzen dira:


a) FX ez beherakorra da.

c) x bakoitzarentzat, FX(x) eskuinetik jarraikia da.


Oharrak:

1. Demagun F: R → R aurreko teoremako hiru propietateak betetzen

dituen funtzioa dela. Orduan, (Ω, F, P) probabilitate-espazioren batean

definituriko X zorizko aldagaia esistitzen da, bere banaketa-funtzioa F

delarik.

2. Banaketa-funtzioak probabilitate-banaketa guztiz zehazten du. Hots, FX

= FY ⇒ PX = PY

3. Banaketa-funtzioaren adierazpen grafikoak ojiba itxura dauka (etenak

egon daitezke).


Teorema: Banaketa-funtzioaren bidez probabilitateen kalkulua:

a) P(X > x) = 1 – FX (x) ∀x∈R

b) P(X < x) = FX (x-) ∀x∈R

c) P(X ≥ x) = 1 – FX (x-) ∀x∈R

d) P(a < X ≤ b) = FX (b) – FX (a) ∀a, b∈R eta a ≤ b

e) P(a < X < b) = FX (b-) – FX (a) ∀a, b∈R eta a ≤ b

f) P(a ≤ X < b) = FX (b-) – FX (a-) ∀a, b∈R eta a ≤ b

g) P(a ≤ X ≤ b) = FX (b) – FX (a-) ∀a, b∈R eta a ≤ b

h) P(X = x) = FX (x) – FX (x-) ∀x∈R (FX-en jauzia x-en)



Propietatea: P(X = x) = 0 ⇔ FX jarraikia x-en
