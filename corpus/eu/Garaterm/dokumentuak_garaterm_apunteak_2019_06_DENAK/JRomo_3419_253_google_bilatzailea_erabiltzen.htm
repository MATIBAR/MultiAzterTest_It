Google bilatzailea erabiltzen

Google bilatzailea erabiltzean komeni da zenbait kontzeptu argi edukitzea. Esate baterako, bilaketa Internet osoan egin nahi den, ala bilatzen ari garen informazioa web gune bakar batean dagoelako esparru estu horretara mugatuko dugun bilaketa.

Internet osoan bilaketak egiten:

Demagun bi hitz bilatu nahi ditugula, bat informatika hitza eta bestea 2013-14 hitza. Horiek idatzita bost miloi t'erdi erreferentzia aurkitzen dira, eta ikusi nola bilatzaileak informatika hitzaren antzekoak hartu dituela ere, adibidez Informtica. Hona hemen bilaketaren irudia:

bi hitz bilatzen

Bost miloi sarrera horien artean ikusi bi hitzak edozein tokitan agertzen direla, eta dakigunean hitz horiek esaldi bat osatzen dutela bilaketa baldintza estuagoa jarriko dugu komatxoak erabiliz, hau "informatika 2013-14" idaztean emaitza askoz zehatzagoa da, aurreko 5.490.000 erreferentzietatik 30.700 erreferentzietara pasatuko gara:

bilaketa zehatza

Bilaketa zehatz hori egitean gure bloga lehen posizioan kokaturik agertzen da. Beraz, komatxoak oso lagungarriak dira bilatzaileak eskaintzen dizkigun emaitzak mugatuagoak eta egokiagoak izan daitezen.

 

Web gune batean bilaketak egiten:

Batzuetan gertatzen da zerbait bilatu nahi dugula web gune jakin batean, hots, web gune jakin baten orrialdeetan. Adibidez, ikus dezagun zenbat orrialde, eta zeintzu orrialde, dituen informatika-2013-14.blogspot.com web guneak une honetan. Bilaketa egitean site: eragilea erabiliko dugu, honelaxe:

bilaketa egin den unean web gune horrek 21 orrialde izan ditu

Demagun web gune horretan gmail hitza bilatu nahi dugula, bilaketaren baldintza idatzi eta 3 orrialdeetan agertzen dela gmail hitza lortuko dugu emaitzatzat:

21 orrialdeetatik soilik 3tan dago gmail hitza

Bilaketan logika alderantzikoa izan daiteke, adibidez zer orrialdeetam ez da gmail hitza agertzen; horretarako -gmail marka erabiliko dugu:

21-3=18 orriealdeetan ez dago gmail hitza

Internet osoan bezala, web gunean ere bilaketa zehatzak egin daitezke komatxoak erabiliz. Ikusi hurrengo bi irudietan zer nolako aldea dagoen gmail kontu bilaketa eta  "gmail kontu" bilaketaren artean:

gmail kontu hitz biak bilatzean

"gmail kontu" hitz bikote zehatza bilatzean
