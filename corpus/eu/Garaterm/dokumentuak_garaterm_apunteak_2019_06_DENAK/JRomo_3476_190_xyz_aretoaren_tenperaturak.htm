XYZ aretoaren tenperaturak

Demagun gela edo areto batean 22 gradu inguruko tenperatura dagoela, gelako puntu guztien tenperaturak ezagunak direla eta bektore batean gordeta daudela zenbaki erreal bezala. Puntu batetik besterako aldea txikia izanik, tenperatura guztiak 22 graduen inguruan daude. Hona hemen aretoaren eskema:

Datu-motak hauek izanik, ondoko bi eragiketak egin:

PROGRAM GelakoTenperaturak ; CONST    X_MAX = 8 ;    Y_MAX = 8 ;    Z_MAX = 8 ; TYPE    tarBolumena = ARRAY [1..X_MAX, 1..Y_MAX, 1..Z_MAX] OF Real ;

Bektorea aleatorioki datuz bete eta bektorearen edukia pantailaratu.

Bektorean informazioa gordetzen eta ateratzen dakigula, ondoko hiru eragiketak egin:

  • Gelako zutabe bat aukeratu eta zutabe horretako punturik hotzena zehaztu
  • Gelako plano bat aukeratu eta plano horretako punturik hotzena zehaztu
  • Gelako punturik hotzena zehaztu

Adibidez, hona hemen programa exekutagarria XYZ_Gela.EXE eta programa horren balizko irteera zati bat:

 

Ariketaren planteamendua ikusita eta programa exekutagarria egikarituz programa zuk zeuk egitea eskatzen da. Lan hori egin ondoren zure programa eta jarraian ematen diren bi programak buruz buru jarri eta bakoitzaren alde onak eta alde ahulak ikusi:

  • zure XYZ_Gela.PAS programa
  • XYZ_Gela1.PAS programa (koordenatu guztiaak Integer datu-motako aldagaiak)
  • XYZ_Gela2.PAS programa (1D, 2D eta 3D koordenatuentzat datu-mota bereziak)
