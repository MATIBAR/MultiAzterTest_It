2.2. Bola irekiak eta itxiak. Esferak.

2.8. Deﬁnizioa.- Izan bitez (X, d) espazio metrikoa, x ∈ X eta r > 0. X-ren ondoko azpimultzoak deﬁni
daitezke:

x zentruko eta r erradioko bola irekia:

B(x, r) = {y ∈ X : d(x, y) < r}.

x zentruko eta r erradioko bola itxia:

B(x, r) = {y ∈ X : d(x, y) ≤ r}.

x zentruko eta r erradioko bola ireki laburtua:

B∗(x, r) = {y ∈ X : 0 < d(x, y) < r} = B(x, r) − {x}.

x zentruko eta r erradioko esfera:

S(x, r) = {y ∈ X : d(x, y) = r}.

2.9. Propietateak.- (X, d) espazio metrikoa izanik:

(i) ∀x ∈ X eta ∀r > 0, B(x, r), B(x, r) = ∅ baina gerta daiteke S(x, r) hutsa izatea.

(ii) 0 < r ≤ s bada, B(x, r) ⊆ B(x, s) eta B(x, r) ⊆ B(x, s).

0 < r < s bada, B(x, r) ⊆ B(x, s) eta S(x, r) ∩ S(x, s) = ∅.

(iii) r > 0 guztietarako, B(x, r) ⊆ B(x, r), S(x, r) ⊆ B(x, r), B(x, r)∩S(x, r) = ∅, B(x, r) = B(x, r)∪S(x, r),

B(x, r) = B(x, r) − S(x, r) eta S(x, r) = B(x, r) − B(x, r).

(iv) {B(x, ri) : i ∈ {1, . . . , n}} zentru bereko n bola ireki badira, orduan ∩ni=1B(x, ri) = B(x, r) da

r = min {ri : i ∈ {1, . . . , n}} izanik.

(v) {B(x, ri) : i ∈ I} zentru bereko bola irekien familia bada eta r = sup {ri : i ∈ I} existitzen bada,

orduan

i∈I B(x, ri) = B(x, r) da.

(vi) {B(x, ri) : i ∈ {1, . . . , n}} zentru bereko n bola itxi badira, orduan ∪ni=1B(x, ri) = B(x, r) da

r = max {ri : i ∈ {1, . . . , n}} izanik.

(vii) {B(x, ri) : i ∈ I} zentru bereko bola itxien familia bada eta r = inf {ri : i ∈ I} = 0, orduan

i∈I B(x, ri) = B(x, r) da.

2.10. Teorema.- (Hausdorﬀ-en propietatea) Izan bedi (X, d) espazio metrikoa. x, y ∈ X eta x = y
badira, orduan existitzen dira bi bola ireki disjuntu bata x puntuan zentratua eta bestea y puntuan, hau da,
∃r, s > 0 non B(x, r) ∩ B(y, s) = ∅ diren.
