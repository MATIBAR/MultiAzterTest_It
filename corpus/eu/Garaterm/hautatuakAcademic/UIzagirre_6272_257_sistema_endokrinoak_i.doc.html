SEINALIZAZIO-SISTEMA BIOLOGIKOAK (Bioeraentzaileka)

Animalien funtzionamendu integratua nerbio-sistemaren eta sistema endokrinoaren kontrol pean dago


-Nerbio-sistema-------Neurotransmisoreak

-Sistema endokrinoa---Hormonak


Zelulen arteko seinalizazio barianteak

• Autokrino: zelula berera

• Parakrino: hurbil dauden zelulen arteko difusioa gune interstizialetik

• Neuronal: sinapsis neuronen artean

• Neuroendokrino: neuronatik zirkulazio sistemara

• Endokrino: Guruin zelulatik odol bitartez itu zelulara


  • Sistema endokrinoak honako prozesu biologiko kontrolatzen ditu:

    • Kolore-aldaketak

    • Ugalketa: sexuaren finkapena, gonadaren garapena,…

    • Hazkuntza

    • Homeostasia: metabolismoa, ormoeraenketa,…

    • Metamorfosia eta muda

    • …


HORMONAK

  • Organo edo ehun batek ekoiztutako substantziak, organismoaren beste eskualde batean jarduteko barne-medioan garraiatu egiten direnak (Jatorria: neurona neurojariatzaileen produktuak)

  • Osaketa kimikoa:

    • Peptidoak: animoazidoak edo péptidos txikiak

    • Lipidoak: esteroideoak batik bat

  • Hormona itu-zelula ezagutu behar du:

    • PEPTIDIKOAK-MINTZ-HARTZAILEAK

    • ESTEROIDEOAK-HARTZAILE ZITOSOLIKOAK

  • Guruin endokrinoetan edo zelula jariatzaile isolatuetan produzitzen dira:

    • GURUIN SOKAKARAK

      • GURUIN FOLIKULARRAK

      • NEURONA NEUROJARIATZAILEAK

    • ZELULA JARIATZAILEAK:

      • JARIAKIN PROTEIKOZKOAK

      • JARIAKIN ESTEROIDEOZKOAK

NEUROJARIAPENA ORNOGABEETAN

  • ANELIDO POLIKETOAK

  • INTSEKTUAK

(Adb., muda, gonada-garapena eta bihotz-erritmoa eraentzeko)

Zerebroa, corpora cardiaca, corpora allata (neotenina), guruin protorazikoa (ekdisona)

  • KRUSTAZEOAK

    • X-organoa (neurojariatzailea, MIH sortu, jariatu)

    • Barrunbe guruina (Neurohemalea, MIH askatu, hemolinfara)

    • Y-organo (endokrinoa, ekdisona sintetizatu)

INTSEKTUEN SISTEMA ENDOKRINOA

  • Hormona diuretikoa

    • Gernuaren ekoizpena kitzikatu

    • Zerebroko zelula neurojariatzaileetan sintetizatu

    • Axonen bidez corpora cardiaca-ra garraiatu

    • Sistema zirkulatoriora askatu (organo nurohemala)

  • Hormona adipozinetiko

    • Gantz-gorputzeko erreserba lipidikoak mobilizatu

    • Corpora cadiaca-ko zelula neurojariatzaileak ekoiztu

    • Sistema zirkulatoriara askatu (organo neurohemala)

  • Hormona protorazikoa (PTTH)

    • Mudaren prozesua eraendu

    • Zerebroko zelula neurojariatzeetan sintentizatu

    • Axonen bidez corpora cardiaca-ra garraiatu

    • Ekdisonaren sisntesia kitzikatu guruin protorazikoan

  • Gazte-hormona (JH)

    • Hazkuntza eta metamorfosia prozesuak eraendu

    • Corpora allata-ko zelula neurohariatzeetan sintetizatu

    • Sistema zirkulatoriara askatu (organo neurohemala)
